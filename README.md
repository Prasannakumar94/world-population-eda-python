# World Population Exploratory Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on a global population dataset using Python to identify trends, patterns, and population distribution across countries and continents.

## Dataset
The dataset contains population statistics of countries including population counts, growth metrics, and geographic information.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Handling missing values
- Descriptive statistical analysis
- Population distribution visualization
- Country-wise and continent-wise comparisons

## Key Insights
- Identified countries with highest population growth.
- Compared population distribution across continents.
- Visualized population trends using bar charts and heatmaps.

## How to Run

Clone the repository:
git clone https://github.com/Prasannakumar94/world-population-eda-python.git
Install dependencies:
pip install -r requirements.txt
Run the notebook:
jupyter notebook
